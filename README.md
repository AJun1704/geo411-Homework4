# geo411-Homework4
This assignment focuses on applying simple linear regression models to analyze relationships between variables and interpret statistical results.
First, we examined whether SAT scores can predict GPA using a linear regression model. The results showed a strong positive relationship, supported by a significant p-value and a high R-squared value.
Next, we analyzed mosquito data to study how the number of development sites affects the number of pupae. We built multiple models, including a linear model, a log-transformed model, and a log-log model. These models helped improve interpretation and address issues such as skewness and variability.
We also evaluated the impact of outliers and compared model performance before and after removing them. Diagnostic plots and histograms were used to assess model assumptions and data distribution.
